# Запуск
Запускайте контейнер и смотрите на результат; в него можно передавать и произвольные аргументы для количества эпох и пожеланий по сохранению json

# Визуализация
В папке с world_viz.html есть пример инициализации и развития мира, можно выбрать его и визуалировать граф отношений сущностей

# Планы (предполагается реализация в указанном порядке)
1. Реализовать агентную систему для валидации генерации или уточнить правила, чтобы не было вопиющих противоречий
2. Написать добротный API для загрузки лора как по схеме, так и по описанию натуральным языком, и рендеринга
3. Использовать графовую БД; 
4. Реализовать агентную систему для генерации квестов
5. Отрисовывать карту с помощью каких-нибудь топологических алгоритмов для деформации краёв при сохранении пространственного расположения биомов;
